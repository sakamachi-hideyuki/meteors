流星と昴の日本神話

 

日本神話には星の神は少ないとされているが、実は数多くの神が流星や昴の神と考えられる。

本書では、火瓊瓊杵尊、素戔嗚尊、饒速日命、大物主神、天津甕星、武甕槌神、建葉槌命など約百柱の神の神名や神話が、流星（火球）や昴に由来すると考えられることを示す。

また、日本神話では和歌などでも用いられる「見立て」（あるものを別のもので表現する技法）が多用されており、次のように解釈できるということを示す。

 

・「流星（火球）は天降る神の姿」「流星は昴から来る」と考えられていた。

・神名に付く「速」は「速い神」つまり「流星の神」であることを意味する。

・神名に付く「櫛」「甕」は「流星（火球）」を意味する。

・神名に付く「玉」「火」「石」「磐」は「星」を意味する。

・五百箇御統、五百箇磐石（湯津石村）、天岩戸は「昴」を意味する。

・スサノオの剣、イザナキの剣は「オリオン座の三つ星」を意味する。

・櫛真智命、天目一箇神、天津麻羅の神名は「昴の神」であることを意味する。

・御食津神の名に付くウカ、ウケ、オカ、オオゲは「大きな器」の意。

・水や雨を司る神の名に付くオカミは「大きな雲の神」、ミツハは「雲」の意。

・神、剣、鏡の名に付くフツは「降る神」つまり「流星の神」を意味する。
